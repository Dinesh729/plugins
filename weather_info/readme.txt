== Description ==

The Weather plugin enables you to get up-to-date weather information for your location. The Weather plugin is a dynamically-generated based on http://www.worldweatheronline.com/



= Languages =

English  



== Installation ==

= You can either install it automatically from the WordPress admin, or do it manually: =

1. Unzip the archive and put the ` weather_info` folder into your plugins folder (/wp-content/plugins/).
2. Activate the plugin from the Plugins menu.
