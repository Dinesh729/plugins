== Description ==

  1. This is a simple facebook login plugin.
  2. just install the plugin and add the login shortcode any where post/page.
  3. Click on FB Login Admin Menu.
  4. Click on General Text and Enter App ID and Secret Key.
  5. Click Save Changes Button. 
  6. Use this shortcode [fb_signin] in your page/post to display "Login With Facebook" button.
  7. Use this shortcode <?php echo do_shortcode('[fb_signin]'); ?> in your page template to display "Login With Facebook" button.
  8. You can check screenshots.

== Installation ==

= You can either install it automatically from the WordPress admin, or do it manually: =

1. Unzip the archive and put the "Facebook Login" folder into your plugins folder (/wp-content/plugins/).
2. Activate the plugin from the Plugins menu.
